HTML cheatsheet
  1. HTML boilerplate code 
    
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title >
</head>
<body >
    this is the body of the html code
</body >
</html >
  2. Heading tags

    There a total of 6 heading tags available in HTML. H1 being the largest and H6 the smallest.


    <H1> tag 

     <h1> Heading 1 </h1>

    <H2> tag 

     <h2> Heading 2 </h2>

    <H3> tag 

     <h3> Heading 3 </h3>

    <H4> tag 

     <h4> Heading 4 </h4>

    <H5> tag 

     <h5> Heading 5 </h5>

    <H6> tag 

     <h6> Heading 6 </h6>

  3. Formatting tags

    Text formatting tags are used to format text or data of HTML documents.

    <b> Tag 

     <b> This tag makes the text bold </b>

    <u> Tag 

     <u> This tag underlines the text </u>
    <i> Tag 

     <i> This tag displays the text in italics </i>

  4. Lists

    Lists can be either numerical, alphabetic, bullet, or other symbols.


    <ol> Tag 

    
<ol> 
<li> Data 1 </li>
<li> Data 2 </li>
</ol>

    <ul> Tag 

    
<ul> 
<li> Data 1 </li>
<li> Data 2 </li>
</ul>

  5. Image

    6. <img> tag


      <img src="Source of image" alt="Alternate text">

  6. Table

    7. A table is a collection of rows and columns. It is used to represent data in tabular form.


    Table Structure 

    
<table> 
<tr> 
<th> Table heading </th>
</tr>
<tr> 
<td> Data 1 </td>
</tr>
</table>
  7. Links

    8. Links are clickable text that can redirect you to some other page.


    <a> tag 

     <a href="www.w3schools.com" > Visit W3schools </a>
