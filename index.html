    <H4> tag

  3. Formatting tags

    Text formatting tags are used to format text or data of HTML documents.

    <B> Tag 

     <B> This tag makes the text bold </B>

    <U> Tag 

     <I> This tag underlines the tex </I>
    <I> Tag 

     <I> This tag displays the text in italics </I>
